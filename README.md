# 리치 마작 설명용 패 배치 도구

https://c-dtd.github.io/Majong-Tabletop-Simulator/

- 하단 마작 패를 클릭해 움직일 수 있는 마작패를 만듬
- 이미지를 화면에 드래그 앤 드롭해서 추가할 수 있음
- 지우고 싶은 요소를 더블클릭 또는 선택 후 delete나 backspace를 눌러 삭제할 수 있음

- 마작 패를 꺼내거나 이동시킬 수 있는 도구로 리치 마작을 설명할 때 도움을 줌
- 리치 마작 외의 용도로도 써도 되지만 리치 마작 패만 있음
- 패 이미지 출처: [ https://github.com/FluffyStuff/riichi-mahjong-tiles ](https://github.com/FluffyStuff/riichi-mahjong-tiles)
